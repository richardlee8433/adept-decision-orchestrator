# ADEPT Decision Orchestrator

## Overview
The ADEPT (Advanced Decision Engine for Payment Transactions) Decision Orchestrator is an enterprise-grade fraud decision orchestration platform designed for high-performance financial environments.

## Key Features
- **Enterprise-Grade Orchestration:** Robust framework for managing complex fraud decisioning workflows.
- **Microsecond Latency:** Built with AsyncIO and FastAPI to ensure ultra-low latency processing for real-time transactions.
- **High-Performance AI Ensemble:** Features an Ensemble Boost Simulation achieving an **81.73% F1-score** and a **0.012% False Positive Rate (FPR)** on the ULB European Credit Card Dataset.
- **Modern UI:** Mastercard-inspired React/Tailwind user interface for intuitive strategy management and monitoring.

## Technical Architecture
- **Backend:** Python, FastAPI, AsyncIO.
- **Frontend:** React, Tailwind CSS.
- **Performance:** Optimized for microsecond-level response times.
